Admin → Whitepapers → Pipeline

- Add future topic ideas
- Reorder queue
- Approve/reject drafts

4. Create a daily cron job:
Run once per day.

- Generate whitepaper draft

- Do NOT publish automatically unless autoPublish is enabled

- subtitle

- PDF/print compatibility
