====================================
P0-1 — AUTO-OPEN MOBILE PROGRAM PANEL
====================================

Problem:
After the first build completes on mobile, users never see the Live Program Panel unless they manually tap the panel button.

Goal:
Immediately reveal the generated program after the first successful build.

Requirements:
- Detect successful first program generation
- On mobile viewports only:
  - automatically open the right-side Live Program Panel
  - animate smoothly
  - preserve existing desktop behavior
- Trigger AFTER streaming completes and program state is hydrated
- Do NOT interrupt during stream
- Do NOT auto-open for every future edit
- Only:
  - first build
  - new full rebuilds

Add:
- lightweight spotlight pulse around the panel for ~2.5s
- one-line helper text:
  "Your training system is ready →"

Files likely involved:
- chat.tsx
- LiveProgramPanel.tsx
- mobile panel state hooks
