==================================================
PART 1 — PREDICTIVE BLOCK ADAPTATION
==================================================

Currently:
- Adaptation happens AFTER sessions

Upgrade to:
- System anticipates future weeks

Add a new function:

buildBlockProjection(profile, recentLogs, currentBlock)

This should:
- Analyze last 7–14 sessions
- Compute:
  - avgDifficulty
  - painTrend
  - adherenceTrend
  - fatigueTrend (derived from duration + difficulty + feeling)

Then modify FUTURE weeks:

IF avgDifficulty > 4 OR fatigueTrend high:
  → reduce Week 2/3 intensity slightly
  → delay intensification
  → extend accumulation

IF adherence < 70%:
  → reduce weekly volume
  → simplify sessions

IF painTrend present:
  → remove risky patterns from FUTURE weeks
  → bias stability + control

IF consistently easy:
  → accelerate progression curve

IMPORTANT:
- This modifies the block STRUCTURE, not just exercises
- Store projection in block metadata
