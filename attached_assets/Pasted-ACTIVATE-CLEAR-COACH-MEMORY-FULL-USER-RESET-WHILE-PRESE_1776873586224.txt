ACTIVATE “CLEAR COACH MEMORY” — FULL USER RESET WHILE PRESERVING SUBSCRIPTION / MEMBERSHIP STATUS

CONTEXT:
On the Settings page there is currently a disabled / “Coming soon” item:
- Clear coach memory

We want to activate this feature.

Expected behavior:
When a user chooses “Clear coach memory,” it should reset them to a clean slate across the TrainChat platform.

That means:
- remove chats
- remove coach memory / learned preferences
- remove saved program history
- remove active program state
- remove session history / logs
- remove user-specific adaptive state
- remove focus-specific learning state
- remove staged learning candidates tied to that user
- clear anything else that makes the system “remember” their past usage

BUT:
- do NOT remove their account
- do NOT remove their auth/login
- do NOT remove their subscription / membership / billing state
- do NOT break access if they are a paying subscriber

This should feel like:
“Start over fresh, but keep my account and membership.”
