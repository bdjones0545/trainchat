Fix the TrainChat “This Week” and exercise logging system so it behaves correctly by session type and exercise type.

CURRENT ISSUES:

1. In the “This Week” tab, tapping “+ more exercises” does not expand the card to reveal the rest of the session.
2. The same warm-up is being repeated for every training day, even when the sessions are different.
3. The exercise logger assumes every exercise uses weight, which is incorrect for jumps, throws, med ball work, sprint drills, mobility work, carries, timed work, and bodyweight movements.
4. Example bug: Broad Jump is prompting for weight entry instead of distance or reps.
5. The current system is treating exercise logging as a uniform strength-lift template instead of matching the actual exercise type.

GOAL:
Fix the “This Week” card expansion behavior, diversify warm-up generation by day/session intent, and make exercise logging dynamic based on exercise category.

----------------------------------------
1. FIX “+ MORE EXERCISES” CARD EXPANSION
----------------------------------------

In the “This Week” tab:

When the user taps “+ more exercises” on a session card:
- the card should expand inline OR open a proper session detail view
- the hidden exercises should render immediately
- the expanded state should be tracked per session card
- tapping again should collapse it if inline expansion is used

Requirements:
- no dead tap targets
- smooth mobile behavior
- preserve current styling
- expansion state should be unique per day/session

Suggested state shape:
expandedWeekCards = {
  [sessionId]: true | false
}

If inline expansion is too messy, route to a dedicated session detail sheet/modal/page.

2. FIX WARM-UP GENERATION LOGIC
----------------------------------------

Do not reuse the exact same warm-up for every session unless the session intent truly matches.

Warm-up generation must be session-aware.

Use session focus to generate the warm-up:
- Lower body force day → lower body mobility, dynamic prep, ankle/hip activation, landing mechanics
- Upper body strength day → T-spine, scap activation, shoulder prep, trunk stiffness
- Full body power day → dynamic total-body prep, sprint/jump/throw readiness, elastic prep

Warm-up rules:
- Each day should have its own warm-up template based on session goal
- Some overlap is okay, but not identical copy/paste warm-ups across all sessions
- Warm-up should include a role label like:
  - Warm-Up
  - Activation
  - Movement Prep

Example:

- hip circles
- inchworms
- glute bridge
- lateral band walk

Upper day:

- shoulder CARs

Power day:

- med ball prep
- snap-down
- dynamic mobility
