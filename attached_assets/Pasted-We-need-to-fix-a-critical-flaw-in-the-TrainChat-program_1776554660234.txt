We need to fix a critical flaw in the TrainChat programming agent:

PROBLEM
The agent is not treating AGE as a real programming constraint. Example:
User input: “I am a 65 year old pickleball player give me a 3 day program”
The system generated things like box jumps, conventional deadlifts at 4–6 reps, pull-ups, and Bulgarian split squats without first applying an age-aware safety and recovery layer.

This makes the output feel like a generic athletic program instead of a context-aware coaching system.

GOAL
Implement a true AGE-AWARE PROGRAMMING LAYER so the agent:
1. Detects age from free text reliably
2. Stores age in structured context
3. Uses age as a hard modifier before exercise selection and ranking
4. Adjusts volume, intensity, exercise risk, impact, and recovery
5. Changes both generation behavior and agent response language

IMPORTANT
Do not just patch the final text response.
Do not just add age to the summary.
Do not fake compliance.
This must affect the real program generation pipeline.

TASK 3 — ADD AGE-AWARE SAFETY / SCALING LAYER
==================================================

Before exercise selection happens, introduce an age-aware decision layer.

Create a function similar to:
buildAgeProfile(age?: number): AgeProfile

Example shape:
{
  bracket: 'under_40' | '40_49' | '50_59' | '60_69' | '70_plus',
  prioritizeJointFriendly: boolean,
  reduceImpact: boolean,
  reduceAxialLoading: boolean,
  emphasizeBalance: boolean,
  emphasizePowerButLowImpact: boolean,
  recoveryAdjustment: 'none' | 'moderate' | 'high',
  defaultIntensityBias: 'normal' | 'conservative',
  defaultVolumeBias: 'normal' | 'reduced'
}

Core rules:
- Age < 50: minimal change
- 50–59: mild joint-friendly bias, mild recovery bias
- 60–69: meaningful reduction in high impact and high spinal load, more stability and controlled power
- 70+: strongly conservative defaults, highly joint-friendly, balance/stability emphasis

For age 60+:
- Reduce high-impact jumping exposure
- Reduce low-rep max-strength bias
- Reduce high spinal loading bias
- Increase unilateral control, balance, trunk stability, rotational control
- Favor “power with low impact” instead of “power with hard landings”
- Favor recoverable training density
- Favor technique quality over aggressive loading

This layer must feed into the real ranking / filtering system.

==================================================

Integrate age into the candidate selection engine in two ways:

A. HARD FILTERS
For users age 60+ by default:
- Block or strongly suppress high-impact plyometrics unless explicitly unlocked
- Block or strongly suppress high-risk landing exercises
- Block or strongly suppress very high spinal loading for low-rep strength prescriptions
- Block or strongly suppress advanced bodyweight strength movements unless experience supports them

B. SCORING ADJUSTMENTS
Add scoring bonuses for:
- joint-friendly exercises
- low-impact power exercises
- supported unilateral work
- balance/stability work
- trunk anti-rotation / anti-extension
- recoverable movement patterns
- rotational work relevant to sport with safe loading

Add penalties for:
- high impact
- high spinal load
- high balance demand without progression path
- poor scalability for older users
- advanced exercises without regression

IMPORTANT:
Do not completely sterilize the program.
For pickleball and similar sports, still preserve:
- rotational power
- lateral control
- unilateral strength
- elastic qualities
But express them in safer forms.

TASK 7 — ADJUST PROGRAM DOSING
==================================================

For age 60+ default programming should also adjust dosing:
- Slightly lower total volume
- Slightly more rest between demanding sets
- Less aggressive loading ranges
- More moderate rep ranges for main strength work unless advanced trainee
- More warm-up/prep quality
- More movement prep for hips, trunk, shoulders, ankles
- Better recovery distribution across the 3-day split

For example:
- Prefer trap bar or goblet patterns over heavy conventional deadlift prescriptions
- Prefer 5–8 or 6–10 type ranges over aggressive 3–5 loading for most cases
- Keep power intent but lower impact
- Include tissue/joint prep appropriate for racket sports and older adults
