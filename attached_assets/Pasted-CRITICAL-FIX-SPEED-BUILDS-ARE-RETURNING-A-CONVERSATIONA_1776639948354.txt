CRITICAL FIX — SPEED BUILDS ARE RETURNING A CONVERSATIONAL PREVIEW INSTEAD OF A FINISHED SAVED PROGRAM

CONTEXT:
In Speed / Footwork mode, the user can request a build (example: "3 days") and after ~60 seconds receives a speed-flavored conversational response like:

- "Here’s the foundation I’m building around you..."
- outlines of Day 1 / Day 2 / Day 3
- mention of exercises that will be included

But no actual full program appears in the right panel and no final saved program is produced.

This means the system is still allowing a conversational / planning response to satisfy a build request instead of requiring a final structured build artifact.

==================================================
PRIMARY GOAL
==================================================

Make Speed build requests complete as real builds:
- full structured program
- parsed successfully
- saved to DB
- shown in the right panel / Active Programs

Do NOT allow a conversational preview to count as a completed build.

TASK 2 — DISALLOW CONVERSATIONAL PREVIEW RESPONSES FOR BUILD INTENTS
==================================================

If resolved intent is:
- BUILD_PROGRAM
- REBUILD_PROGRAM
- any explicit build action

Then the response must NOT be allowed to end as:
- a high-level explanation
- a preview
- a scaffold
- a “here’s the foundation”
- a “I’ll include...”
- a “once you complete your profile”

For build intents, only two valid outcomes are allowed:
1. full structured program build saved successfully
2. clean recoverable error

No in-between conversational completion.

==================================================

Strengthen buildSpeedResponseContract() so the model is explicitly forbidden from returning:
- prose-first build plans
- preview language
- future-tense promises
- “I’m building around you”
- “I’ll map this out”
- “once you complete your profile”

Add an explicit rule like:

"For build intents, return ONLY the final structured JSON build artifact. Do not return an outline, preview, explanation, or promise of a later build."

==================================================

- "this will be structured"

If detected during a build intent:

- either retry with stricter build-only instruction or fail cleanly
